💬 HOW TO USE CHAT:

METHOD 1: Via Live Collaboration (Recommended)
───────────────────────────────────────────────
1. Click "Collab" button in header
2. Click "Create New Room" or "Join Room"
3. Enter your username when prompted
4. ✅ Chat connects automatically!
5. ✅ Chat panel opens automatically!
6. Start coding and chatting together!

METHOD 2: Via Chat Panel Directly
──────────────────────────────────
1. Click chat icon (💬) in left activity bar
2. Enter your username
3. Enter room code (or click "Create" to generate one)
4. Click "Join" or "Create"
5. Start chatting!

✨ KEY FEATURES:

🔒 Room Isolation
   • Messages never cross rooms
   • Each room has its own chat
   • 12-character room codes

👤 Username Uniqueness
   • No duplicate usernames per room
   • Smart suggestions if name is taken
   • Auto-generate alternatives (Alice2, SwiftAlice, etc.)

🔔 Notifications
   • Red badge on chat icon for new messages
   • Shows unread count
   • Browser notifications (with permission)

🔄 Auto-Integration
   • Create/join collaboration → chat connects automatically
   • Same room code for both features
   • Disconnect from collab → chat disconnects too

🐛 TROUBLESHOOTING:

Problem: "Failed to connect to server"
Solution: Make sure backend is running (node server.js)

Problem: "Username already taken"
Solution: Select one of the suggested alternatives or enter a different name

Problem: Chat icon not visible
Solution: Check left activity bar (below Files and Import icons)

Problem: Messages not appearing
Solution: Verify both users are in the same room (check user list)

Problem: Room code not working
Solution: Ensure code is exactly 12 characters and host is still connected
