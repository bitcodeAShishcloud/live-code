Quick Guide

Features

Multi-Language: HTML, CSS, JS, Python, JSON, Markdown
Live Preview: Real-time code execution
Console: View logs, errors, and outputs
Theme: Dark/Light mode toggle
Collaboration: Real-time code sharing
Export: Download as ZIP or individual files

Desktop Guide

  • Sidebar: Click Files icon to open/close sidebar
  • New File: Click "+ New File" button in sidebar
  • Rename File: Double-click file name in tab to rename
  • Import: Click Import icon or drag files into editor
  • Run Code: Click Run button or press Ctrl/Cmd + S
  • Resize: Drag the vertical bars to resize panels
  • Export File: Hover on tab, click download icon
  • Console: Click terminal icon to toggle console
  • Console Input: Type JavaScript at bottom of console and press Enter

Mobile Guide

  • Sidebar: Tap floating button (bottom-left) to open files
  • New File: Open sidebar → tap "+ New File"
  • Run Code: Tap Run button in header
  • Switch Mode: Toggle Manual/Auto run mode
  • Scroll: Swipe horizontally on header buttons
  • Console: Tap terminal icon to show/hide console
  • Export: Tap download icon on file tabs

Keyboard Shortcuts

Ctrl/Cmd + S Run Code
Ctrl/Cmd + Enter Run Code
F2 Rename File
Ctrl/Cmd + Shift + F Format Code
Ctrl/Cmd + Shift + P Toggle Preview
Ctrl/Cmd + B Toggle Sidebar
F5 Refresh Preview
Ctrl/Cmd + Z Undo
Ctrl/Cmd + Shift + Z Redo

Tips & Tricks

  • Use Auto Run mode for instant preview updates
  • Type JavaScript in Console Input and press Enter to execute
  • Use Code Snippets for quick templates
  • Share code in real-time with Collaboration
  • Export entire project as ZIP for backup
  • Format code automatically with Format button
  • Double-click resizers to reset panel sizes
  • Double-click file names in tabs to rename them
